# ML-MCDM Project Objectives

## Objective 1: Current Data Index Calculation (2011-2024)
Calculate final composite index scores and rankings for 63 provinces using 29 sub-criteria (8 criteria) scores (2011-2024) with MCDM methods combined with Evidential Reasoning (ER).

## Objective 2: Future Year Forecasting (2025)
Forecast index scores for 63 provinces using historical data (2011-2024, 14 years) with advanced state-of-the-Art Ensemble Learning combined via Meta-learner and apply multiple MCDM methods (combined with Evidential Reasoning (ER)) to predicted 2025 final composite index scores.
