# ML-MCDM Project Objectives

## Objective 1: Historical Performance Assessment (2011-2024)
Quantify provincial governance performance across a 14-year panel for Vietnam's 63 provinces. This involves a hierarchical 7-phase pipeline using 29 sub-criteria (8 criteria), objective CRITIC weighting, and a consensus-based ensemble of 5 MCDM methods plus a Raw Sum baseline.

## Objective 2: Future Performance Forecasting (2025+)
Project future governance outcomes using the production-hardened TIER 3 ensemble forecasting architecture. This system leverages a 4-model Super Learner (CatBoost, Bayesian Ridge, SVR, ElasticNet) trained on the 14-year historical panel, with calibrated uncertainty quantification via Conformal Prediction.

## Objective 3: Decision Support & Sensitivity Analysis
Provide robust decision-making artifacts, including rank-stability heatmaps, temporal stability metrics, and multi-level weight sensitivity analysis to ensure the technical and mathematical integrity of the performance assessment.

---
**Status**: Production-Hardened (v2.0)
**Dataset**: PAPI Vietnam (2011-2024)
